🗑️ Suppression de vos données — Vidygo

Dernière mise à jour : 5 juin 2026

Supprimer vos données depuis l'application

Vidygo stocke toutes vos données localement sur votre appareil. Vous pouvez les supprimer à tout moment directement depuis l'application :

📱 Étape 1 — Ouvrez Vidygo
⚙️ Étape 2 — Appuyez sur le menu ⋮ en haut à droite → Paramètres
🔄 Étape 3 — Section Application → Réinitialiser l'application
✅ Étape 4 — Confirmez la suppression

Toutes vos vidéos, playlists et préférences seront définitivement supprimées de votre appareil.

Désinstaller l'application

Désinstaller Vidygo supprime également toutes les données locales associées à l'application.

Données publicitaires (AdMob)

Vidygo utilise Google AdMob pour afficher des publicités. Google peut collecter certains identifiants publicitaires. Pour gérer ou supprimer ces données :

  • Réinitialisez votre identifiant publicitaire Android : Paramètres Android → Confidentialité → Publicités
  • Consultez : Politique de confidentialité Google
