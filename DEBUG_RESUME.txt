🔧 SI ÇA NE FONCTIONNE PAS
═════════════════════════════════════════════════════════════════════════════

❌ Émulateur ne démarre pas:
   → Vérifier 4GB RAM disponible
   → Vérifier 3GB espace disque libre
   → Redémarrer l'ordinateur

❌ ADB ne voit pas l'émulateur:
   $ adb kill-server
   $ adb start-server
   $ adb devices

❌ Erreur de build:
   $ ./gradlew clean
   $ ./gradlew build

❌ Android Studio désynchronisé:
   File → Sync Now
   Puis relancer debug
