🚀 DÉMARRAGE RAPIDE
═════════════════════════════════════════════════════════════════════════════

  1️⃣ Synchroniser le projet Gradle:
     ./gradlew sync

  2️⃣ Ouvrir dans Android Studio:
     File → Open → C:\Development\Android\Vidygo

  3️⃣ Créer/démarrer un émulateur:
     Tools → AVD Manager → Ou connecter un device

  4️⃣ Exécuter l'application:
     Run → Run 'app' (ou Shift+F10)

  5️⃣ Tester les fonctionnalités:
     ├─ Voir écran avec état vide
     ├─ Ajouter une vidéo
     ├─ Ouvrir/Partager/Supprimer
     └─ Navigation fluide
